Smart Parking Detection

COMPUTER VISION ML IMAGE & VIDEO PROCESSING

Description

Computer vision system that automatically detects free vs occupied parking spaces in surveillance video. It processes video frames in real time, applies classification, shows visual feedback (bounding boxes), and generates an annotated output video.

Key Features

  • Define parking slot locations via a predefined mask (image / region of interest).
  • Preprocess images & video frames: mask cropping, image preprocessing (cleaning, normalization), optionally dimensionality reduction (PCA) to understand dataset separability.
  • Train a classifier (binary classification: empty vs occupied) using the prepared dataset.
  • Real-time detection: process video, compare frames, apply classification only when a significant change between frames is detected (to avoid redundant computation).
  • Visualize results: bounding boxes (green for free slots, red for occupied ones), display counts (free vs total), show annotated video live.
  • Save annotated video output with overlays & results.

Tech stack

  • Language: Python
  • Image & Video Processing: OpenCV, scikit-image
  • Machine Learning / Classification: scikit-learn (PCA + classifier)
  • Numerical & Data Handling: NumPy, Matplotlib (for visualization and output)

Impact / Outcomes:

  • Real-time detection of parking occupancy with annotated visualization — useful for monitoring & managing parking lots.
  • Reduces manual monitoring; can be used in smart city dashboards or IoT systems.
  • Provides both live feedback and recorded annotated video for auditing or further analysis.

Moroccan Political Insight

BIG DATA STREAMING LLM SOCIAL MEDIA ANALYTICS

Description

A real-time Big Data pipeline to monitor, classify and analyze political sentiment on Twitter in Morocco. Tweets are streamed, cleaned, classified using a large language model (LLM), and visualized via an interactive dashboard to reveal trends, opinion distribution, and keyword insights. The system enables understanding of support vs criticism toward parties and leaders.

Workflow

  • Data Ingestion : Collect raw tweets from social media and store them on Firebase Realtime Database.
  • Streaming & Cleaning : Stream data from Firebase into a Kafka topic; process the stream using Spark Structured Streaming; clean text (remove noise, special characters etc.); store cleaned data into Hadoop HDFS
  • Export Clean Data : Export cleaned JSON from HDFS to local storage; transfer files (for example via WinSCP) for further processing
  • LLM-Based Classification : Use Mistral 7B (in Google Colab with GPU) to classify tweets (support vs criticism or other political labels).
  • Visualization & Dashboard : Build a Streamlit app to display interactive insights: sentiment distribution by party, keyword frequencies, trend over time

Tech stack

  • Data Ingestion / Storage: Firebase Realtime Database
  • Streaming & Processing: Kafka (with Zookeeper), Apache Spark Streaming, Hadoop HDFS
  • LLM : Mistral 7B for classification; python notebooks (Colab)
  • Dashboard : Streamlit
  • Utilities: JSON data, WinSCP for file transfers

End-to-End Job Intelligence System

PYTHON WEB SCRAPING ORACLE DATABASE ML PDF PARSING GUI

Description

Built a complete data pipeline and intelligence system to analyze the job market and assist job seekers. The workflow integrates multi-source web scraping, data cleaning, structured storage in Oracle DB, interactive dashboards for analytics, predictive modeling for future job/skill demand, and a CV matching engine to recommend suitable positions.

Workflow

  • Data Collection : Scraped job postings from multiple sources (emploi.ma, jobboom, marocannonces, remoteok)
  • Data Cleaning & Normalization : Preprocessing with regex & text normalization (remove accents, unify labels).
  • Data Storage : Structured job data stored in Oracle SQL database.
  • Data Visualization : Connected to Oracle DB, created interactive dashboards with PyQt5 + Matplotlib to visualize statistics of jobs & skills.
  • Predictive Analytics : Trained ML pipeline (StandardScaler + XGBoost Regressor) to forecast in-demand jobs and skills for the upcoming year.
  • CV Matching : Users can upload a CV (PDF), the system extracts skills via pdfminer, compares them with job requirements, and recommends best-fit roles.

Tech Stack

  • Programming: Python
  • Scraping & Cleaning: Requests, BeautifulSoup, regex, unicodedata
  • Database: Oracle SQL
  • Visualization: PyQt5 (QtWidgets), Matplotlib (FigureCanvasQTAgg)
  • ML & Forecasting: scikit-learn (StandardScaler, Pipeline), XGBoost (XGBRegressor)
  • Document Parsing: pdfminer (extract_text)
  • Data Handling: Pandas

Impact

  • Forecasted future demand for jobs and skills to guide students & professionals.
  • Enhanced user experience with personalized job recommendations via CV matching.
